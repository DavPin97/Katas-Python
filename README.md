# Proyecto: Katas Python

Este repositorio contiene una serie de Katas de Python que han servido para asentar las bases del lenguaje aplicando todos los conceptos vistos en clase.

## Contenido del Proyecto

- **Archivo principal:** `katas_python_Final.py` - Contiene todos los enunciados y sus soluciones.

## Pasos Seguidos Durante el Proyecto

### 1. Análisis de los Ejercicios
- He revisado cada ejercicio, he repasado las clases practicas y teoricas para finalmente determinar cuales eran las mejores soluciones.

### 2. Implementación en Python
- He aplicado las soluciones usando estructuras de control, funciones y manipulación de datos.
- He optimizado el código para hacerlo más eficiente y legible.

### 3. Pruebas y Validación
- He comprobado cada función con diferentes entradas para asegurarme de que devolvía los resultados esperados.
